# Password Checker & Generator

## Project Description
This project is a simple Password Checker and Generator built using Python. 
It helps users check the strength of their passwords and also generate strong, secure passwords automatically.

## Features
- Checks password strength (Weak, Medium, Strong)
- Validates length and character types
- Generates random secure passwords
- Easy to use command-line interface

## Technologies Used
- Python
- Random module
- String module

## Why This Project?
This project demonstrates basic Python concepts like functions, loops, conditionals, and string handling.
